# Style Assist

The AI Style Assist enables users to digitize their closet and eliminate the struggle that comes with picking an outfit, allowing users to feel more confident. Use our application in just three easy steps: Upload, Generate, and Favourite. AI Style Assist includes features such as sorting through your uploaded clothes by colour, type of clothing, and formality. Use the chatbot that specializes in generating outfits with prompts or inspiration photos online. Become more comfortable with yourself through embodied cognition, improving your mood, productivity, and simplifying your lifestyle.
 

## Developing

### Backend

```
cd backend
uv venv # (only run this once; if ur venv has not been created)
source ./.venv/bin/activate
uv pip install -r requirements.txt
```

### Frontend

```
cd web
pnpm i
pnpm dev
```
